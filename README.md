 🎲 Dice Roll Game

A simple JavaScript Dice Roll Game that generates random dice values.

This project allows users to roll one or multiple dice and displays both:
- The dice images
- The numeric results

 Features

- Roll 1 or more dice
- Random dice values (1–6)
- Displays matching dice images
- Shows the rolled numbers
- Simple and clean UI

 Built With

- HTML
- CSS
- JavaScript


 How It Works

1. Enter the number of dice you want to roll.
2. Click the **Roll Dice** button.
3. The app:
   - Generates random numbers between 1 and 6
   - Displays the corresponding dice images
   - Shows the results on the screen

You can roll 2 dice (or more) at the same time!
